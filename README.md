Objetivo da extração de dados em diversos locais utilizando especialmente a Biblioteca Selenium, a qual é destinada para automação de testes e interações automatizadas com navegadores da web, permitindo simular interações humanas com um navegador; a biblioteca Re, utilizada para trabalhar com expressões regulares, em especial no script foi utilizada para remover caracteres não alfanuméricos e prefixos antes de algumas palavras; a biblioteca Pandas para tratamento e manipulação dos dados extraídos.
